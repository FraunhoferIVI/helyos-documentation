The helyOS Framework
====================

**helyOS is a framework for accelerating the development of yard automation projects.** helyOS has a microservice architecture tailored for applications in fleet management of autonomous driving vehicles in delimited areas. It provides guidelines for the domain-driven design of applications in yard automation.  The software core works as a ready-to-use backend for a Control Tower and is able to orchestrate microservices and dispatch assignments for complex missions.
